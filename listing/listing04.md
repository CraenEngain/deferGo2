Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Создается небуферизованный канал ch для передачи значений типа int
В анонимной функции, которая запущенная как горутина, запускается цикл, который 10 раз отправляет значения i в канал ch.
После идет итерация по каналу ch. Эта конструкция продолжает получать значения из канала до тех пор, пока канал не будет закрыт.
В коде нет механизма для закрытия канала ch. Когда все 10 значений отправляются в канал, анонимная горутина завершает выполнение, но канал остается открытым.
Необходимо добавить close(ch) для странения deadlock'а.
```
