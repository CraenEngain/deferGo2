Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Функция test()
Возвращаемое значение функции test() обозначено как x int.
В теле функции используется defer, который выполняется после того, как основное тело функции завершает выполнение, но перед тем, как функция вернет результат.
В defer определяется анонимная функция, которая увеличивает x на единицу.
Затем x присваивается значение 1.
После этого используется return, который инициирует возвращение значения. Но до того, как значение будет возвращено, defer увеличивает x на единицу.
Значение x в момент возврата равно 1, но затем увеличивается на 1.
Функция вернет 2.
Функция anotherTest()
Здесь x является обычной переменной.
defer используется для увеличения x на единицу.
Переменной x присваивается значение 1, и затем она возвращается.
В отличие от функции test(), defer изменяет x, но изменение происходит уже после того, как значение x было сохранено для возврата.

```
